Uhm, wad dat?

Good you're asking.
To get straight to the point, this dApp proofs that your Ethereum Account is acctually an EAO (or Externally Owned Account) and NOT a smart contract. Let that sink in for a moment.

Why build this tool?

Now, you and your friends obviously know the difference between a user and a smart contract. Other smart contracts, though, have troubles telling them apart. This results in a more-complex-than-necessary user experience, as well as unnecessary development burdens.

What is this useful for?

Soulbound Tokens for example. When only EAOs are allowed to hold some token, it is possible to make them actually non transferable. Imagine one could buy a degree in some subject without going to university. Sounds stupid, but without a system like Outsider unavoidable on Ethereum.

How does Outsider help?

To verify that some address is in fact an EOA, developers need to recover a public key from a so called ECDSA signature. While not as complicated as it sounds, this is not exactly trivial to implement.
Outsider hides this logic and thereby reduces the difficulty to a singular, copy-and-pastable, line of code.
This means multiple things:

A) (New) Users are a little bit less overwhelmed by all this arguably cryptic software.

B) The end product is more secure.

C) Developers can spend more time on actually building innovative software.
